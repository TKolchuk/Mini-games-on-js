## 🎯 О проекте

**Mini-Games Hub** — это одностраничное веб-приложение, объединяющее шесть разноплановых мини-игр, реализованных на чистом JavaScript. Проект создан как демонстрация навыков работы с DOM, обработки пользовательского ввода и создания интерактивного интерфейса без использования сторонних библиотек.

**Цель проекта**: показать, что увлекательный игровой опыт можно создать с помощью нативных возможностей браузера, обеспечив при этом высокую производительность и мгновенную загрузку.

## 🕹️ Функции

### Доступные игры:

| № | Игра | Описание |
|---|------|----------|
| 1 | 🔢 Угадай число | Компьютер загадывает число от 1 до 100, игрок угадывает с подсказками «больше/меньше» |
| 2 | 🧮 Простая арифметика | Генерация случайных примеров на +, −, ×, ÷ с проверкой ответа |
| 3 | 🔤 Переверни текст | Введите любой текст — приложение отобразит его в обратном порядке |
| 4 | ✂️ Камень, ножницы, бумага | Классическая игра против компьютера с рандомным выбором |
| 5 | ❓ Простая викторина | Три вопроса с вариантами ответов и подсчётом правильных ответов |
| 6 | 🎨 Генератор цветов | Клик по кнопке меняет фон страницы на случайный HEX-цвет |

### Особенности интерфейса:
- ✅ Полностью адаптивный дизайн (десктоп / планшет / мобильные)
- ✅ Плавные анимации и hover-эффекты на чистом CSS
- ✅ Интуитивная навигация через якорные ссылки
- ✅ Декоративные SVG-элементы для визуального обогащения
- ✅ Отсутствие внешних зависимостей — всё работает офлайн
